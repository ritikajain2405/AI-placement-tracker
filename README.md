# 🚀 AI Placement Tracker

A full-stack web application built using Django to help students efficiently track their placement preparation journey.

## 🔹 Features
- Track DSA topics with progress monitoring
- Manage company applications (Applied / Selected / Rejected)
- Interactive dashboard with real-time insights
- Mock interview tracking with feedback
- Clean and responsive UI

## 🛠 Tech Stack
- Backend: Django
- Database: SQLite
- Frontend: HTML, CSS, Bootstrap

## 🚀 Future Enhancements
- AI-based recommendations
- Resume analysis using ML
- Interview performance prediction

## ⚙️ Setup Instructions

1. Clone the repository:
https://github.com/ritikajain2405/AI-placement-tracker/tree/main

3. Navigate to project folder:
cd AI placement Tracker

4. Create virtual environment:
python -m venv venv

5. Activate environment:
venv\Scripts\activate   (Windows)

6. Install dependencies:
pip install -r requirements.txt

7. Run migrations:
python manage.py migrate

8. Start server:
python manage.py runserver

9. Open in browser:
http://127.0.0.1:8000/
